我的2017

2018-01-03 • 标签: 随笔

2016年12月5日，我走进大活304、306面试。算下来，2017是我在团队度过的第一个完整的年头。

前端的学习之路

刚开始跟着新人任务学 HTML 和 CSS，那时候自己还蛮搞笑的，像高中生一样在小本本上面记笔记。然后度过了这辈子最痛苦的春节，倒不是任务很多，现在想想就是自己给自己的压力特别大，每天就看JS、Backbone.js，连C语言都没怎么学会，上来就接触这些，再加上各种Web的开发里的术语，那时候感觉很难受。记得第一次搭环境，我怎么也弄不好，厚着脸皮烦了导师一整天，所以很感激她那时候耐心的帮助我。

没想到冬令营的五天更加艰难，什么都不会，动不动就跑去问，除了UI其他代码都是老板带（代）我写的。每次都是，“我不会了…”，然后他把我推开，扶了扶眼镜，啪啪啪敲好，把电脑扔回给我。要展示的时候，他比我们还开心，还跟我说“你们是写的最完整的”。

大一下开始做自己的第一个项目木犀通行证，导师带着我写，老板给我 Code review，还有每周的分享会，才真正的让我从一无所知开始了解Web前端。暑假的时候开始写木犀内外的项目，每天在进度簿里面记录写了多少 bug，解决了 bug 之后也知道了更多。

从最开始 Backbone.js 让我知道 MVC， 学 MVVM 框架 Vue 注意到数据逻辑和样式要分离，现在学 Vuex 状态管理，构建工具用 Webpack，编辑器也 Sublime、Atom 转向 VS Code。接下来巩固基础的同时，想了解更多 Service Workers、PWA 等新技术吧，还有就是深入学好 JavaScript。

我和团队的大家

大一下，16级一起做Lab，我是最水的那个，起步很晚，学的踉踉跄跄很吃力，看着大家非常强心里蛮虚的，可是没有人嫌弃我，还记得szy同学耐心的给我讲一个lab讲了三遍2333。CSAPP 和 C++，自己学的很不牢固，所以接下来要再复习很多很多遍。

暑假一起去上海参加hack.init()，组织者和参赛者都非常年轻，却是我经历过最豪华的hackathon。上海科技大学的宛如高新科技园，建筑和设备都崭新先进，还有很多可爱的猫！（是橘猫没错了）比赛期间不停地吃吃喝喝拿贴纸衣服，晚上八点就有美丽的小姐姐给我们送好吃的蛋糕当宵夜。不仅有紧张刺激的开发，更重要的是让我们看到了同龄人，甚至是低龄的小朋友们如此优秀，他们不仅有创意还有实力，十岁开始编程而不减热情的大有人在，那时候才让我感受到什么是祖国的花朵，互联网行业的未来啊。

紧接着夏令营一起写项目、忙招新，16级的我们渐渐凝聚在一起，共同进步，互相鼓励。

最后一点想说的

2017 年，最开始很茫然慌张，之后感觉最好的状态在暑假，用心的写木犀博客、刷pat。然后这学期除了为团队和组内做了一点点事情，项目修修补补、刷了一下LeetCode，总体很怠惰，非常惭愧。2018 要改变这种消极的状态，数据结构、算法、JS这些基础打牢，把项目做好，复习 CSAPP 学习计算机基础，多刷题。

真挚的感谢团队的每一个人，特别是老板、婉颖学姐、雅静学姐、皖莹。

新的一年，自己变得成熟了，开始承担更多的责任，希望不辜负你们的期望，团队和我能够越变越好吧。
